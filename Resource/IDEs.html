дҪҝз”ЁcondaжҢҮд»ӨдҫҶе®үиЈқжҲ–жӣҙж–°еҘ—д»¶#

йҖҡеёёдёҖеҖӢзЁӢејҸиӘһиЁҖзҡ„еҹ·иЎҢжңғе•ҹеӢ•е…¶й җијүзҡ„еҹәжң¬еҘ—д»¶пјҢи®“дҪ еҸҜд»ҘдҪҝз”Ёи©ІзЁӢејҸиӘһиЁҖзҡ„еҹәжң¬еҠҹиғҪпјҢеҰӮиіҮж–ҷзөҗж§ӢгҖҒжөҒзЁӢжҺ§еҲ¶пјҢд»ҘеҸҠйғЁеҲҶзҡ„ж•ёеӯёеҮҪејҸгҖӮиҖҢиҰҒеҒҡжӣҙеӨҡзҡ„з§‘еӯёиЁҲз®—гҖҒиіҮж–ҷеҲҶжһҗжҲ–ж©ҹеҷЁеӯёзҝ’е°ұиҰҒд»°иіҙз¬¬дёүж–№жүҖй–Ӣзҷјзҡ„еҘ—д»¶пјҢйҖҷдәӣеҘ—д»¶еҰӮnumpyгҖҒmatplotlibгҖҒpandasгҖҒbeautifulsoup4гҖҒgensimзӯүпјҢйғҪйңҖиҰҒе…Ҳе®үиЈқеңЁйӣ»и…ҰдёҠпјҢдёҰеңЁзЁӢејҸзўјеҹ·иЎҢеүҚпјҢеҠ ијүе…ҘзҸҫдёӢзҡ„еҹ·иЎҢз’°еўғжүҚеҸҜд»ҘдҪҝз”ЁйҖҷдәӣеҘ—д»¶жүҖжҸҗдҫӣзҡ„еҮҪејҸгҖӮAnacondaжҸҗдҫӣдәҶдёҖеҘ—дҫҝеҲ©зҡ„е®үиЈқж–№ејҸпјҢеҸҜд»ҘеңЁMacдёҠзҡ„TerminalжҲ–WinдёҠзҡ„CMDзҡ„е‘Ҫд»ӨеҲ—ијёе…ҘcondaжҢҮд»Өе°ұе®үиЈқжүҖйңҖиҰҒзҡ„еҘ—д»¶гҖӮжүҖи¬Ӯе®үиЈқзҡ„жҰӮеҝөжҳҜжҠҠи©ІеҘ—д»¶еҫһйҒ з«Ҝз¬¬дёүж–№дјәжңҚеҷЁдёҠијүеӣһдҪ зҡ„йӣ»и…ҰдёҰе®үиЈқеңЁдҪ йӣ»и…ҰиЈЎпјҢдҪҶйҖҷдёҚд»ЈиЎЁдҪ зҡ„зЁӢејҸе·Із¶“еҸҜд»Ҙй–Ӣе§ӢдҪҝз”Ёе®ғдәҶпјҢжҜҸж¬ЎдҪ иҰҒеҹ·иЎҢи©ІзЁӢејҸзҡ„жҷӮеҖҷпјҢйғҪйӮ„еҝ…й ҲиҰҒијүе…ҘпјҲimportпјүе®ғжүҚеҸҜд»Ҙй–Ӣе§ӢдҪҝз”ЁгҖӮи«ӢиҰӢд»ҘдёӢAnacondaзҡ„е®ҳж–№иӘӘжҳҺдәҶи§ЈиҰҒеҰӮдҪ•з”Ёcondaе®үиЈқиҲҮжӣҙж–°pythonеҘ—д»¶гҖӮhttps://conda.io/docs/user-guide/getting-started.html

    Copy/paste the URL into your browser to open the jupyter notebook

еҝ«йҖҹйҚөjupyter notebook#

  • Shift + Enter run the current cell, select below

  • Ctrl + Enter run selected cells

  • Alt + Enter run the current cell, insert below

  • Ctrl + S save and checkpoint

  • Enter take you into edit mode

  • D, D (press the key twice) delete selected cells

  • Esc take you into command mode

  • Y change the cell type to Code

  • M change the cell type to Markdown

  • A insert cell above

  • B insert cell below
